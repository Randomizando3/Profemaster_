Como criar e gerenciar aulas

Siga o passo a passo abaixo para criar uma aula com capa e materiais. Cada passo mostra exatamente o que fazer.

Dica: você pode vincular a aula a uma Instituição e Turma (opcional) para manter tudo organizado.

Passo 1 — Acessar aulas

Abra a lista de Aulas

No menu principal, toque em Aulas para visualizar as aulas cadastradas.

Passo 2 — Criar aula

Toque em + Nova

Na tela de aulas, toque em + Nova para criar uma nova aula.

Passo 3 — Preencher os dados

Preencha instituição, turma, título e duração

Escolha (opcional) a Instituição e Turma, digite o Título da aula e a Duração. Se quiser, selecione uma Capa e escreva a Descrição.

Instituição/Turma: opcional Capa: selecionar/remover Descrição: conteúdo/observações

Passo 4 — Anexar via link

Adicione materiais por link

Cole links de vídeos, documentos ou conteúdos externos para ficar tudo vinculado à aula.

Exemplo: link de YouTube, Drive, PDF online, site de referência, etc.

Passo 5 — Upload de arquivos

Envie arquivos do dispositivo

Faça upload de PDFs, imagens ou outros arquivos de apoio diretamente no seu celular.

Os arquivos ficam salvos como materiais da aula.

Passo 6 — Salvar

Finalize tocando em Salvar

Revise os dados e toque em Salvar para concluir a criação da aula.

Pronto. Aula criada e pronta para uso ✅
